Molecular Descriptors and Fingerprints in RDKit

Molecular Descriptors

Molecular descriptors are numerical representations of chemical structures and properties. RDKit provides a wide range of descriptors that can be used for various applications, such as QSAR modeling, virtual screening, and property prediction.

Calculating Molecular Descriptors

You can calculate molecular descriptors using the Descriptors module in RDKit. Hereâ€™s an example:

from rdkit import Chem
from rdkit.Chem import Descriptors

# Create a molecule from SMILES
mol = Chem.MolFromSmiles('c1ccccc1')

# Calculate descriptors
mw = Descriptors.MolWt(mol)
logp = Descriptors.MolLogP(mol)
tpsa = Descriptors.TPSA(mol)

print(f"Molecular Weight: {mw:.2f}")
print(f"LogP: {logp:.2f}")
print(f"Topological Polar Surface Area: {tpsa:.2f}")
Molecular Weight: 78.11
LogP: 1.69
Topological Polar Surface Area: 0.00

LogP, also known as the partition coefficient, is a crucial parameter in chemistry, particularly in drug discovery and development. It represents the logarithm of the partition coefficient of a solute between octanol and water at near infinite dilution. LogP is a measure of the preference of a compound to dissolve in either water or an organic solvent, such as octanol, when uncharged.

RDKit provides many other descriptors, including:

  • Descriptors.NumRotatableBonds(mol): Number of rotatable bonds
  • Descriptors.HeavyAtomCount(mol): Number of heavy atoms
  • Descriptors.NumHAcceptors(mol): Number of hydrogen bond acceptors
  • Descriptors.NumHDonors(mol): Number of hydrogen bond donors
  • Descriptors.MolWt(mol): Molecular weight
  • Descriptors.MolLogP(mol): LogP value
  • Descriptors.TPSA(mol): Topological polar surface area

You can explore the complete list of 208 available descriptors from this Greg Landrum blog post.

Molecular Fingerprints

Molecular fingerprints are binary or count-based representations of molecular structures, used for similarity searching, clustering, and other cheminformatics tasks.

Generating Molecular Fingerprints

RDKit supports various types of fingerprints, including Morgan fingerprints, RDKit fingerprints, and MACCS keys. Hereâ€™s an example of generating and comparing Morgan fingerprints:

from rdkit import Chem, DataStructs
from rdkit.Chem import AllChem

# Create molecules from SMILES
mol1 = Chem.MolFromSmiles('c1ccccc1')
mol2 = Chem.MolFromSmiles('c1ccccc1C')

# Generate Morgan fingerprints
fp1 = AllChem.GetMorganFingerprintAsBitVect(mol1, 2, nBits=2048)
fp2 = AllChem.GetMorganFingerprintAsBitVect(mol2, 2, nBits=2048)

# Calculate Tanimoto similarity
similarity = DataStructs.TanimotoSimilarity(fp1, fp2)
print(f"Tanimoto Similarity: {similarity:.2f}")

Output:

Tanimoto Similarity: 0.86

In this example, we generate Morgan fingerprints with a radius of 2 and a bit vector size of 2048 for two molecules. We then calculate the Tanimoto similarity between the fingerprints, which measures the similarity between the molecular structures.

You can also generate other types of fingerprints, such as RDKit fingerprints and MACCS keys:

# RDKit fingerprint
rdkit_fp = Chem.RDKFingerprint(mol, maxPath=5)

# MACCS keys
maccs_fp = Chem.MACCSkeys.GenMACCSKeys(mol)

Fingerprint Operations

RDKit provides several functions for working with fingerprints, including:

  • DataStructs.FingerprintSimilarity(fp1, fp2, metric=DataStructs.TanimotoSimilarity): Calculate fingerprint similarity using different metrics (e.g., Tanimoto, Dice, Cosine)
  • DataStructs.BulkTanimotoSimilarity(fps, bulkfps): Calculate Tanimoto similarities between a set of fingerprints and a bulk set of fingerprints
  • DataStructs.FingerprintDensity(fp): Calculate the density (fraction of bits set) of a fingerprint
  • DataStructs.UnionBitVects(fp1, fp2): Compute the union of two fingerprints
  • DataStructs.IntersectionBitVects(fp1, fp2): Compute the intersection of two fingerprints
  • DataStructs.FoldFingerprint(fp, nbits): Fold a fingerprint to a smaller size

These operations enable various applications, such as similarity searching, clustering, and fingerprint analysis.
